# Дело №001: «Час, которого не было»

Интерактивный детектив, в котором посетитель сам расследует вымышленное преступление —
не угадывает наугад, а исключает подозреваемых через противоречия в алиби, уликах и
хронологии.

> **Двое часов показывают разное время. Один из них лжёт.**

В ночной часовой мастерской «Меридиан» найден убитым её владелец. Напольные часы на месте
преступления остановились на 21:15, и всё указывает, что тогда всё и случилось. Но наручные
часы на руке жертвы разбились на 22:52. Кто перевёл время — тот и убийца.

## Как это работает

Расследование идёт через шесть экранов:

| Экран | Механика |
|---|---|
| **Доска улик** | Пробковая доска с 12 карточками. Клик открывает улику в крупном плане; оттуда можно протянуть SVG-нить к подозреваемому — это карта версий игрока |
| **Подозреваемые** | Четыре досье с мотивом и алиби |
| **Допрос** | Ветвящийся диалог. Каждый ответ сверяется с уликой: алиби либо подтверждается, либо **противоречит** — противоречие подсвечивается |
| **Хронология** | Drag & drop на 9 событий. Время скрыто — порядок восстанавливается логикой. Ошибка даёт подсказку «не сходится по времени» |
| **Обвинение** | Финальный выбор: подозреваемый + мотив + улика-доказательство |
| **Разгадка** | Полный разбор дела — но только если обвинение верное. Ошибся — ответа не будет, возвращайся к уликам |

### Дедукция — это механика, а не декорация

Улики логически ведут к **одному** виновному через исключение. Ложные следы допустимы, но
каждый опровергается другой уликой: отпечатки подмастерья на орудии объясняются его
ежедневной работой, сильнейший мотив наследницы разбивается о камеру банкомата, а знаток
часов с явной обидой на жертву в момент убийства находился в другом городе. Ни одна деталь
не требует догадки.

## Стек

- **React 18 + Vite 6** — состояние расследования общее для всех экранов (собранные улики,
  нити, ответы допроса, порядок хронологии, обвинение), поэтому React оправдан
- **Роутинг — стейт-машина** в React Context, без `react-router`: поток почти линейный
- **Нити на доске** — SVG с замером геометрии узлов через `ResizeObserver`
- **Анимации** — чистый CSS (открытие папки на `rotateX`, «прикалывание» карточек,
  прорисовка нитей, зерно плёнки, мерцание лампы)
- **Прогресс** — `localStorage`, бэкенд не нужен
- Зависимостей ровно четыре: `react`, `react-dom`, `vite`, `@vitejs/plugin-react`

### Визуальный стиль

Нуар: чёрно-белая база и **один** акцент — янтарный (свет лампы). Печатная машинка
(Special Elite / Courier Prime) для «документов», чистый sans-serif для интерфейса.
Зернистость плёнки, виньетирование, курсор-лупа на интерактивных элементах.
Акцент вынесен в CSS-переменную `--accent` — меняется одной строкой.

Учтён `prefers-reduced-motion`: зерно, мерцание и анимации отключаются.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # статика в dist/
npm run preview  # предпросмотр сборки
```

Бэкенда нет — на выходе статический сайт, который кладётся на любой хостинг.
Роутинг на состоянии, а не на URL, поэтому SPA-redirect'ы не нужны.

## ⚠️ Спойлеры

`content/case-001.md` и `src/state/caseData.js` содержат **полное решение дела** —
кто убийца, почему и какая улика его изобличает. Если хотите расследовать сами,
сначала пройдите дело, а потом читайте исходники.
